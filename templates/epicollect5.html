How you can contribute to our project?

The dataset of Environmental Noice Pollution measurements that used in this project comes from Epicollect5 and contains data collected in an specific area of the municipality of Pune, India. Epicollect5 is a mobile and web application for free and easy data collection. Projects are created by using the web application at five.epicollect.net, then downloaded to the device to perform the data collection. It provides both the web and mobile application for the generation of forms and freely hosted project websites for data collection. If you are interested to be a contributor to the dataset that we used for this project, you can easily download the Epicollect5 application and, by the instruction that provided here you can record your data in the dataset.
